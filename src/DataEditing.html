About The Software

The Electric Vehicle Analyzation Interface, a.k.a EVAI is a software designed to assist the Dual Motor Test Rig. The Dual Motor Test Rig is an electric vehicle test rig having two BLDC motors and a common central loading system. The rig was designed to allow students to experiment and analyze the working of the dual motor system found in electric vehicles. EVAI allows users to compare experimental data obtained from the Dual motor test rig with the actual data. There are 8 windows, each of them representing a curve that was obtained from the data collected from the test rig. The actual data is already available in the software which the users may use to compare with the data obtained by them from test rig. You will be able to enter your data by editing the ‘Obtained data’ column. To do so, the following steps are done:
The window shows below is designed for the Efficiency vs Speed Characteristics.

  1. Select the cell you wish to edit using your mouse pointer. The number present in the cell will then be displayed on the INPUT cell.
  2. Replace the data in the INPUT CELL with the obtained data.
  3. Click ‘Edit’.
  4. Click ‘Generate Chart’.
